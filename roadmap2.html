Zero Robotics High School Roadmap

Week 1 – Game Start
• Kick-off the competition
• Review rules & objectives
• Brainstorm early strategies
Week 2 – Initial Strategy
• Begin coding basic functions
• Explore strategies in simulation tool
• Assign team roles (coding, testing, docs)
• Submit check-in code + write-up
Week 3 – Midpoint Check-in
• Receive feedback from MIT mentors
• Refine strategies and fix bugs
• Identify strengths & weaknesses
• Collaborate to iterate together
Week 4 – Finalizing Code
• Debug and optimize algorithms ⚡
• Write final strategy explanation
• Test code on multiple layouts
• Prep for final submission
Week 5 – Final Submission
• Submit final code + documentation
• Double-check requirements
• Celebrate team progress
• Student Reflections
Week 6 – Preparing Finals
• Celebrate accomplishments
• Review ISS Finals format
• Watch Final Tournament with Friends and Family!
